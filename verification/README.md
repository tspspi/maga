# Verification scenarios

This directory contains a set of simple verification scenarios to validate
the correctnes of the implementation.

## Single Coil

For a single coil two coil configuration have been simulated and compared
with analytical solutions of the Biot-Savart law. The results are presented
in ```res/sc.pdf```

## Helmholtz Pair

Simple Helmholtz Pairs are simulated and compared to analytical solutions
in ```res/helmholtz.pdf```
